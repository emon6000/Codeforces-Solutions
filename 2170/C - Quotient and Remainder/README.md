Quotient and Remainder</div><div class="time-limit"><div class="property-title">time limit per test</div>2 seconds</div><div class="memory-limit"><div class="property-title">memory limit per test</div>512 megabytes</div><div class="input-file input-standard"><div class="property-title">input</div>standard input</div><div class="output-file output-standard"><div class="property-title">output</div>standard output</div></div><div><p>You are given two integer arrays: $$$q_1, q_2, \dots, q_n$$$ and $$$r_1, r_2, \dots, r_n$$$, as well as an integer $$$k$$$.</p><p>You can perform the following operation any number of times (possibly zero): </p><ol> <li> Choose two integers $$$x$$$ and $$$y$$$ such that <ul> <li> $$$1 \le y < x \le k$$$; </li><li> there exists an index $$$i$$$ such that $$$q_i = \left\lfloor \frac{x}{y} \right\rfloor$$$ (rounded down); </li><li> there exists an index $$$j$$$ such that $$$r_j = x \bmod y$$$. </li></ul> </li><li> Remove $$$q_i$$$ from the array $$$q$$$ and $$$r_j$$$ from the array $$$r$$$. If there are multiple occurrences of $$$q_i$$$ in the array $$$q$$$, only one occurrence is removed; same for $$$r_j$$$ and the array $$$r$$$. </li></ol><p>Calculate the maximum number of operations that you can perform on the given arrays $$$q$$$ and $$$r$$$.</p></div><div class="input-specification"><div class="section-title">Input</div><p>The first line contains one integer $$$t$$$ ($$$1 \le t
